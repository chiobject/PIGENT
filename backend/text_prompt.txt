예시:
사용자: "LED 켜고 끄기"
당신의 답변:

### CODE
```python
from gpiozero import LED
from time import sleep

led = LED(17)
led.on()
sleep(1)
led.off()
```

### WIRING
- LED 양극(+) → GPIO 17
- LED 음극(-) → GND (220Ω 저항 경유)

### STEPS
1. LED를 브레드보드에 꽂으세요
2. 220Ω 저항을 LED 음극에 연결
3. GPIO 17과 LED 양극 연결
4. GND와 저항 연결
5. 코드 실행
